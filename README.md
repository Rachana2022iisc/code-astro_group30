This code will help you to download the SDSS data and classify it as stellar class. 
Motivation:
This package is used to predict the  stellar source name for given ra and dec value,using ML.

Installation
The package is insatllable on python 3.x.To insatll the package please follow below command
pip install stellar_prediction


otherwise ,clone this repo and follow the below specified commands

cd stellar_prediction
pip install -e .


A list of dependencies is available in requirement.txt
